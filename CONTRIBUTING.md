# Contributing to Nova-Sortable 5

Thanks for wanting to contribute!

## How to Contribute

1. **Fork** the repository
2. **Create a branch** for your fix or feature (`git checkout -b feature/your-feature`)
3. **Commit** with clear messages
4. **Push** your branch
5. **Open a Pull Request**

## Guidelines

- Keep your code clean and consistent with the existing style
- Add comments where it helps others understand your changes
- If your PR fixes a bug, link the related issue
- Test before you submit

## Communication

For big ideas, please open a **discussion or issue** first to align direction.
