# Stable v1.0.0

🚀 **Stable Release v1.0.0**

Nova-Sortable 5 is now production ready!

- Drag-and-drop reordering via SortableJS directly from the index view
- Up/down arrow buttons for precise single-step ordering
- Visual feedback: green border on moved row, red border on displaced row
- Toast notifications on success/error
- No full page reload — instant DOM updates
- Auto-assigns order values on model creation
- Publishable config for global defaults (`frontbyte-nova-sortable.php`)
- Fully compatible with Laravel Nova 5.x and PHP 8.2+
- Optimized for both light and dark mode
- Built with Vue 3 Composition API + TypeScript

🔗 Demo: https://demo.novafront.dev
🧩 Requirements: Laravel 10–12, Nova 5.x
